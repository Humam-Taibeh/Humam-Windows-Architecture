# HT Core Architecture

An advanced, modular Windows optimization and deployment framework designed for power users.

## 🚀 Features
- **Smart Deployment:** Automated software installation with dependency verification.
- **Safety-First:** Automated system restore points and session-based registry backups.
- **System Optimization:** Performance tweaks, telemetry removal, and service management.

## 🛠️ Usage
1. Download or clone the repository.
2. Run `start.bat` as Administrator.
3. Follow the menu prompts to deploy or optimize your system.

## 🛡️ Safety
This tool modifies system settings and registry keys. Always ensure you have a backup or a recent system restore point.
